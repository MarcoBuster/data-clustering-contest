Realme X2 gets Dark Mode and Nightscape for selfie camera with the latest update

15 Nov 2019 by Sagar

Realme has released a new software update for the Realme X2 in Europe which brings in system-wide dark mode to the mid-ranger. And with this, the total number of Realme smartphones having dark mode reach 11, which include the Realme 1, U1, 2 Pro, 5, 5 Pro, 3, 3i, 3 Pro, X, and XT.

The new firmware bumps up the Android security patch level on the Realme X2 to October 2019, but more importantly, it brings Nightscape Mode for the selfie camera.

Additionally, the new build, bearing version number RMX1993EX_11.A.15, comes with an updated charging animation and now lets you expand the notification center with a swipe down gesture anywhere on the home screen.

Here's the complete changelog of the update:

  • Security

    • Android security patch: October 2019

  • Settings

    • Added Dark Mode
    • Added realme Laboratory
    • Added a switch of swipe down on home screen for global search or notification center
    • Added a switch of Notification Dot for updates
    • Removed the reminder after the hotspot enabled
    • Removed the prompt while rooted
    • Updated charging animation, showing more details

  • Camera

    • Added nightscape of front Camera
    • Optimized partial AF out of focus

The update should reach all the Realme X2 units in Europe in a few weeks, and if you can't wait for it to arrive on your device, you can install it manually by downloading the package from the source link below.
